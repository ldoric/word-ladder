# Word Ladder

A Python project for building and experimenting with word ladder data and logic.

## Current Status

This repository is in early setup. The first notebook currently in the project is:

- `notebooks/01_word_lists_combine.ipynb`

## Project Setup

1. Create and activate a virtual environment:
   - Windows (PowerShell):
     ```powershell
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
     ```
2. Install dependencies (add packages as your project grows):
   ```powershell
   pip install jupyter
   ```

## Run Notebook

```powershell
jupyter notebook notebooks/01_word_lists_combine.ipynb
```
